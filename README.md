# Health Coach System
Personal AI-powered health and fitness coaching system.
## Components
- **Decision Engine**: AI-powered daily planning
- **ETL**: Data ingestion from Intervals.icu and SparkyFitness
- **Telegram Bot**: Daily briefings and Q&A
- **Database**: PostgreSQL for all health data
## Environments
- **Test**: `healthcoach-test` (192.168.1.50)
- **Production**: `healthcoach-prod` (192.168.1.51)
